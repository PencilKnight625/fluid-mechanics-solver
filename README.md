# Fluid Mechanics Solver

A Flask-based web application for solving fluid mechanics problems.

## Features
- Reynolds Number Calculator
- Flow Rate Calculator
- Bernoulli Equation
- Darcy-Weisbach Equation
- Pipe Flow Tools

## Tech Stack
- Python (Flask)
- HTML (Jinja Templates)
- C++ (planned integration)

## How to Run

pip install -r requirements.txt
python app.py
